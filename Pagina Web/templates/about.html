Analisis accesible de grupos y caracteristicas en Datos de Potenciales Pacientes de Diables utilizando técnicas de aprendizaje no supervisado

Esta propuesta de Trabajo de Fin de Máster (TFM) aborda un novedoso enfoque para trabajar con los datos de potenciales pacientes de diabetes. En lugar de usar la tradicional predicción, este TFM propone la aplicación de técnicas de aprendizaje no supervisado como el análisis de conglomerados con el objetivo de identificar patrones y grupos dentro de los pacientes. Junto a ello se plantea la creación de una interfaz web que permita de forma accesible introducir datos de pacientes para clarificarlos en alguno de los grupos identificados. En los últimos años el área del aprendizaje no supervisado se ha desarrollado considerablemente, existiendo numerosos e innovadores algoritmos complejos que permiten el clustering de los datos de forma eficiente y efectiva. En cuanto a la metodología seguida, el proceso incluiría un análisis exhaustivo del conjunto de datos, la selección y evaluación de varios algoritmos de clustering, y la optimización de los mismos para obtener resultados precisos y significativos. Para dar un enfoque completo y práctico al proyecto se propone la creación de una web que podría ser una herramienta valiosa para médicos y pacientes. En la fase final del trabajo se presentará un resultado de los diferentes algoritmos de clustering aplicados a los datos, así como un ejemplo de aplicación real de los mismos desde la plataforma web. Para el desarrollo del trabajo se utilizará Python como lenguaje de programación y diferente software de libre acceso: Jupyter Notebook, Visual Code.
